Welcome to REST SQL API

Notice to my fellow developers

This API allow developers to make SQL request using HTTP request: POST for INSERT, GET for SELECT, PATCH and PUT for UPDATE and finally DELETE for DELETE. This API is totally 100% 10/10 RESTful. It means that I theorically shouldn't tell you how to use it and you should guess but since I don't agree with this shit so here it is: 

POST https://localhost/tablename
Input in JSON:
[
  {
    "col1": "val1",
    "col2": "val2",
    "col3": "val3"
  },
  {
    "1col1": "1val1",
    "1col2": "1val2",
    "1col3": "1val3"
  }
]
Output in JSON:
{
  "id": [
    "1",
    "2"
  ],
  "uri": [
    "/tablename/1",
    "/tablename/2"
  ]
}
GET https://localhost/tablename[/id] Output Format: JSON 
PUT https://localhost/tablename[/id] Input Format: JSON 
PATCH https://localhost/tablename[/id] Input Format: JSON 
DELETE https://localhost/tablename[/id]

This API is secured by unique API keys stored in a SQL table, if the API key checking is active, you should provide the API key as the value of the 

Authorization
header of your request. Don't forget that when using HTTPS headers are also encrypted and thus make this secure.
